
Энэхүү нээлттэй эхийн төсөл нь Андройд хөгжүүлэгчдийн апп бүтээхэд хэрэглэгдэх ерөнхий хэрэг болох функцүүдийг агуулсан болно.  
LibraryProject тул дараах зааврын дагуу холбож ашиглана уу:
Өөрийн хийж буй Project-ийн Properties хэсэг рүү орж Android гэсэн хэсгийг сонгоод Library хэсэгт энэхүү ToolsLibrary-г нэмж өгсөнөөр ашиглах боломжтой болно. 


Эхний байдлаар бэлдсэн тул цөөхөн хэдэн класс байгаа ба цаашид үргэлжлүүлэн хөгжүүлэх болно. 

Одоогын байдлаар ашиглаж болох боломжууд:
1. Url-р дамжуулж өгөгдөл унших(уншсан өгөгдөл String төрөлтэй байх ба ямар object болгож ашиглах нь таны таалал). AsyncTask дотор хийж өгсөн байгаа тул шинээр ямар нэг Thread үүсгэх шаардлага байхгүй
2. Интернэт орчинд байгаа зургыг ImageView-д харуулах. 
3. HTML өгөгдөл дотороос img таг-уудыг түүж авах.
